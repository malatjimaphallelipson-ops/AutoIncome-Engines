# AutoIncome Engines‚Ñ¢ API

Automation for Income ‚Äî Make Money Work for You  
A Freedom Wheels‚Ñ¢ Platform

## Overview

This API powers the AutoIncome Engines SaaS platform, enabling:

- User authentication
- Engine creation and lifecycle management
- Automation activation
- Learning Hub access
- Payment session creation
- Admin controls

## Setup

1. Import `autoincome-engines-openapi.json` into Postman.
2. Import Postman Collection.
3. Import Environment file.
4. Set base_url to production or local.
5. Authenticate and retrieve JWT token.
6. Add token to environment variable.

- Auth
- Engines
- Payments
- Learning Hub
